## 更新内容

- 发布首个稳定版 Windows 桌面客户端，提供面向迁移场景的备份、校验与恢复流程。
- 支持扫描已安装软件、个人文件目录和便携软件候选，并在备份前提供可审查的预览结果。
- 支持将个人文件与确认后的便携软件打包为单文件 `.wrh` 归档，同时仅记录已安装软件清单，不备份安装型软件本体。
- 支持读取归档、校验归档完整性，并按类别或按具体根目录选择恢复范围。
- 恢复前会显示恢复摘要，包括已选根目录、预计文件数和预计恢复大小；恢复默认避免覆盖已有文件，也可显式启用“跳过已存在文件”模式。
- 修复 Windows 发布版额外弹出命令行窗口的问题，并修复界面中文文案与中文字体显示异常。
