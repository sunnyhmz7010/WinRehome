## 更新内容

- 修复 Windows 发布版同时弹出命令行窗口和图形界面的问题，测试版现在默认只显示 GUI。
- 修复界面中文文案异常与方框字问题，启动时会优先加载 Windows 自带的中文字体，并优先使用更接近 Windows 默认观感的中文字体顺序。
- 便携软件扫描支持目录型候选和单文件 `.exe` 候选，并过滤明显的安装器/卸载器文件，减少误报。
- 便携搜索根目录不再只写死少数盘符，会按当前机器实际存在的 Windows 盘符动态收集常见便携目录。
- 桌面界面改成更清晰的工作区结构，提供总览、扫描计划、归档恢复三类页面，并增加快速筛选、命中统计、批量选择和更明确的选中态。
- 总览页增加快速入口与状态摘要，扫描页和恢复页补充空状态与结果卡片，整体更适合做测试版演示和实际试用。
