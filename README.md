# WinRehome

WinRehome 是一个面向 Windows 的迁移备份工具原型，目标是在尽可能节省备份空间的前提下，保留真正有迁移价值的个人数据。

## 项目定位

- 已安装软件只记录清单，不备份程序本体。
- 便携软件作为候选项识别，并和个人文件一起打进单一归档文件，而不是一个软件一个压缩包。
- 个人文件通过白名单目录和迁移价值规则筛选，不采用“只要不在 `Program Files` 就全备份”的粗放方式。
- 默认排除缓存、临时文件、日志、构建产物等可重建内容。

## 当前技术栈

- Rust 1.94+
- `eframe/egui`：轻量原生桌面 UI，不依赖 WebView2
- `winreg`：扫描 Windows 注册表中的已安装软件信息
- `walkdir`：文件系统遍历与候选目录发现

## 当前已实现

1. 已安装软件注册表扫描与清单预览
2. 更克制的高价值用户目录收集与默认推荐
3. 便携软件候选识别、置信度判断与默认选择策略
4. 可审查的桌面端备份计划预览，包含文件数与体积估算
5. 生成单文件 `.wrh` 归档，内含安装软件清单、已选目录/便携软件信息与文件数据
6. 读取最近一次 `.wrh` 归档并恢复到目标目录

## 后续里程碑

1. 备份计划持久化与用户自定义规则
2. 单文件归档格式与元数据索引
3. 压缩、校验与恢复流程
4. 增量扫描与变更跟踪
5. 更完整的迁移向导体验

## 发布说明

- GitHub Release 发布后会自动触发工作流构建 Windows 版本。
- 工作流会在 Release 资产中上传 `WinRehome-vX.Y.Z-windows-x64.exe` 和对应的 `sha256` 校验文件。

- 归档内容校验/恢复向导的更完整 UI
- 备份计划持久化与用户自定义规则
- 锁定文件与更强的增量跟踪支持
